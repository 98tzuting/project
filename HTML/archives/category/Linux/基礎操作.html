Linux — 基礎指令

: Posted on 2022-08-28

相信各位點進來想必對Linux指令有一點需求
像是 ls、mkdir、cd、ssh、scp、cp ……
只需要用過幾次就不會忘記的 common sense
首先要先了解什麼是絕對位置和相對位置
通常你的位置會在目錄也就是波浪號 ~
最開始位置為 /
有波浪號或是..就是使用相對位置，一個點代表現在位置，兩個點代表上一層
絕對位置通常很大一串，像是 /home/使用者/project/test1/justlike.py
使用絕對位置會比較麻煩複雜，但在腳本中建議使用絕對位置以免檔案抓不到
但一般在Linux處理使用相對位置較快速。

這裡我使用為Ubuntu 來做示範

指令 作用 備註
ls 顯示位置底下的資料 有許多額外的附加條件
像是 -a 列出所有項目包括隱藏的 -l 直式列出
mkdir 創建資料夾
cd 移動 後面可接絕對位置或是相對位置
ssh 遠端連線 ssh 使用者@IP位置
cp 複製
scp 遠端複製 rsync 是另一種方式，速度較快可定期備份和壓縮傳輸
mv 移動or改名字 移動時後面加路徑，改名字則為文字
rm 刪除 加入 -r 可以刪除資料夾 -f 可以強制刪除
vi 開啟某檔案 有插件 vim 可以客製化的使用，更方便
touch 創建資料夾
pwd 查看現在位置
